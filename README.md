# The Room - 学徒之盒

一个基于 WebGL/Three.js 的 3D 解谜游戏，灵感来自经典的 The Room 系列。

## 游戏简介

你收到了一个包裹，里面是一个精致的木盒，附带一封信：

> "亲爱的学徒，如果你能打开这个盒子，说明你已经准备好接受真正的训练了。——你的导师"

这是你的第一个考验——打开学徒之盒。

## 玩法说明

- **旋转盒子**：拖动鼠标旋转观察盒子的每一面
- **缩放**：滚轮放大查看细节
- **点击**：点击可互动物体
- **物品栏**：右下角显示收集的物品，点击选中后点击目标使用

## 谜题提示

1. 仔细观察盒子的每一面
2. 有些机关需要先解除阻碍才能操作
3. 收集物品后思考它们的用途

## 技术栈

- Three.js (WebGL 3D 渲染)
- Vite (构建工具)
- 原生 JavaScript (ES6+)

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署到 GitHub Pages

```bash
npm run deploy
```
